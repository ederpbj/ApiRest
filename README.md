## Comandos iniciais

| Descrição            | Comandos                                           |
| -------------------- | -------------------------------------------------- |
| GIT                  | **\_**                                             |
| Git ApiRest          | `Git clone https://github.com/ederpbj/ApiRest.git` |
| Criar pasta api      | `cd api`                                           |
| CMD                  | **\_**                                             |
| Iniciar o projeto    | `npm init -y`                                      |
| Dependências         | `npm install express mongoose cors`                |
| Restartar automático | `npm install nodemon -D`                           |
